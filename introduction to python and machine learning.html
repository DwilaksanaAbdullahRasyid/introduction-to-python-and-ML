Parameter/argumen pada algoritma/class K-Nearest Neighbor¶

  • algorithm = ball_tree, kd_tree, brute, auto
  • leaf_size = Akan berfungsi apabila ball_tree atau kd_tree dipilih. Mempengaruhi kecepatan query data(tergantung memory juga)
  • metric = https://en.wikipedia.org/wiki/Minkowski_distance
  • metric_params = dict
  • n_jobs = int
  • n_neighbors = int
  • p = 1 : rumus manhattan distance, 2 : rumus euclidean distance(default)
  • weights = str/callable, uniform(default)
In [24]:
knn.fit(x,y)
knn.score(x,y)
Out[24]:
0.98
  • fit() = Digunakan untuk membuat sebuah model pembelajaran mesin. Jadi si machine itu learning'nya ya lewat fungsi ini
  • score() = Digunakan untuk menguji akurasi model
